# Plumbing

Simple Sidkiq.cr server that implements a worker. The jobs are queued by a Ruby
web server.

## Installation

```
$ make
```

## Usage

To start the Sidekiq server: `./sidekiq`

To start the Sidekiq Web server: `./web`

## Testing

To run the test suite: `make test`
